# ORT Tracker

Tracker de avance para Ingeniería en Sistemas (ORT Uruguay), Plan 2019.

---

## ¿Qué hace?

- Mostrá el plan de materias de Ingeniería en Sistemas (Plan 2019)
- Marcá materias como no cursada, cursada o aprobada
- Visualizá qué materias están habilitadas y cuáles tienen requisitos pendientes
- Identificá materias estratégicas para avanzar más rápido

## Features

- Carrera disponible: Ingeniería en Sistemas (Plan 2019)
- Seguimiento de progreso de carrera
- Requisitos por materia (previas y cantidad de aprobadas)
- Materias estratégicas con impacto de desbloqueo
- Modo claro y modo oscuro

## Setup local

1. Cloná el repositorio:
	```bash
	git clone https://github.com/ftavares19/OrtTracker.git
	cd OrtTracker
	```

2. Ejecutá el frontend:
	```bash
	cd frontend
	npm install
	npm run dev
	```
